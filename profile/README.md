## Numaproj Contrib

This is a community contribution organization where users share their [Numaflow](https://numaflow.numaproj.io/) sources, sinks, user-defined functions (UDFs), and other extensions.

Most repositories here are community-maintained. The following are maintained by the numaflow-developers themselves:

- [kafka-java](https://github.com/numaproj-contrib/kafka-java) - Kafka source and sink in Java
- [apache-pulsar-java](https://github.com/numaproj-contrib/apache-pulsar-java) - Apache Pulsar source and sink in Java

Contributions are welcome - feel free to share your own sources, sinks, and UDFs with the community.
